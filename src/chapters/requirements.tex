\section{Functional Requirements}
\label{sec:functional_requirements}

This chapter presents a comprehensive analysis of the requirements governing the DQN-based adaptive traffic signal control system. It covers functional, non-functional, hardware, and software requirements that collectively define the expected behavior, performance constraints, and operational environment necessary for successfully simulating and optimizing traffic flow at the Thapathali intersection using SUMO and reinforcement learning, with the aim of outperforming conventional fixed-timer signal control plans.

\begin{table}[H]
    \centering
    \small
    \caption{Functional Requirements for the Project}
    \label{tab:functional_requirements}
    \begin{tabular}{cp{12cm}}
        \toprule
        \textbf{ID} & \textbf{Description} \\
        \midrule
        FR-001 & The system shall import the Thapathali intersection map from OpenStreetMap and configure it in \gls{sumo} with realistic traffic patterns. \\
        FR-002 & The system shall implement a \gls{dqn}-based agent capable of changing traffic signal phases to optimize traffic flow. \\
        FR-003 & The system shall update the \gls{dqn} target network periodically to facilitate stable real-time learning. \\
        FR-004 & The system shall accept traffic state inputs including vehicle counts, queue lengths, and waiting times. \\
        FR-005 & The system shall output traffic light control commands via \gls{traci} based on the \gls{dqn} agent's decisions. \\
        FR-006 & The system shall log key performance metrics (\eg queue length, throughput) for post-simulation analysis. \\
        FR-007 & The system shall visualize simulation progress via \gls{sumo}-GUI for monitoring purposes. \\
        FR-008 & The system shall allow comparative evaluation against baseline fixed-time or actuated signal plans. \\
        \bottomrule
    \end{tabular}
\end{table}

\section{Non-Functional Requirements}

\section{System Requirements}
\label{sec:system_requirements}

\subsection{Hardware Requirements}
\label{subsec:hardware_requirements}
\begin{itemize}
    \item \textbf{Minimum:} 64-bit CPU, 8 GB RAM, 10 GB free disk space.
    \item \textbf{Recommended:} 4+ core CPU, 16 GB RAM, SSD storage.
    \item \textbf{Optional (Training):} Dedicated GPU with 4+ GB VRAM or Google Colab access.
\end{itemize}

\subsection{Software Requirements}
\label{subsec:software_requirements}
\begin{itemize}
    \item \textbf{Operating System:} Windows 10+, MacOS or Ubuntu 20.04+.
    \item \textbf{SUMO:} Eclipse \gls{sumo} 1.26.0 or above.
    \item \textbf{Python:} 3.8 or higher.
    \item \textbf{Key Libraries:} \gls{traci}, pandas, PyTorch, NumPy, Matplotlib.
    \item \textbf{Development Tools:} VS Code, Git, and GitHub.
\end{itemize}
